# capstone
## list of files
`control.py` - test script that displays live video feed over wifi and allows control of drone via keyboard inputs
### controls
  - `return` = takeoff and hover
  - `q`      = land
  - `w`      = forward
  - `s`      = backwards
  - `a`      = left
  - `d`      = right
  - `e`      = up
  - `c`      = down
  - `z`      = rotate left
  - `x`      = rotate right
